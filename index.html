FAQs

Q: How do I use the AutoSkip Extension?
A: Install the extension from the Chrome Web Store and it will automatically skip intros and recaps on supported streaming sites.

Q: Which sites are supported?
A: The extension currently supports Netflix, Amazon Prime Video, and Peacock.
